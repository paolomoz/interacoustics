This is how to use the VRA feature in your Equinox suite.

Start by opening the VRA Pure software.

This step is required for the VRA window to activate.

In the VRA Pure software, you can create your own playlists.

It includes new playlists showing dinosaurs at different stages.

An egg appears, hatches slowly, and eventually reveals a dinosaur.

This game-like approach aims to engage the older pediatric patients during VRA.

Once you're in the suite, you can select the playlists you've created.
