This is how you perform a SISI test on the AD528.

Press and hold the test button and select the SISI test.

Select 'tone' or 'warble' on Channel one.

If cross hearing is present, activate masking on Channel two.

Select a desired test frequency and set the input level 20 dB above threshold using the scroll wheel on the audiometer.

In the most common type of SISI test, the increment steps are set to 1 dB.

You can change the increments by pressing and holding the function button ‘AM’.

Prior to presenting the actual test of 1 dB increments, a trial of two or five-dB steps can be employed to ensure that the patient understands the test procedure and what will be required.
