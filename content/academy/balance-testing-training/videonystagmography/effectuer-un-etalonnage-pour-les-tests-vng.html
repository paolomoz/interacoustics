Dans cette vidéo, vous découvrirez le processus d'étalonnage, qui est une étape essentielle du VNG pour mesurer avec précision les mouvements oculaires.

Le calibrage est une étape essentielle en VNG. Afin de mesurer avec précision les mouvements oculaires, le système VNG doit savoir à quelle distance les yeux se trouvent de la caméra et jusqu'où ils doivent se déplacer pour regarder une cible de position connue. Ceci est réalisé en passant par le processus d'étalonnage, qui est détaillé et démontré dans cette vidéo.
