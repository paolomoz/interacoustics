Touch Keyboard Support

This is how to get started setting up and using the Touch Keyboard.

The Touch Keyboard comes with two rotary dials on the sides and a touch screen in the middle. Around each rotary dial, there are buttons for your thumbs, as well as buttons on the top of the rotary dials for your index finger.

The Touch Keyboard is powered on by holding the two top buttons. When the Touch Keyboard is on, you can see the battery status and connection status on the top.

In the top menu, you can access the device settings. You can adjust the brightness, turn the wireless connection on or off, and set a sleep timer for the screen.

Connect the Touch Keyboard to the Equinox or Affinity Suite by inserting a USB-C cable at the top of the device. Next, connect the USB-C cable to the PC. With the cable in place, launch the Affinity or Equinox Suite.

Once you are connected to the suite, you can open your top menu again to select the test you want to perform. This list can be configured in the software suite. Once the keyboard has been connected to the suite for the first time, it can run wirelessly.
